# GIF Generator for macOS - 需求规格说明书 (Specs)

## 1. 项目概述 (Project Overview)
一款专为 macOS 设计的轻量级表情包转换工具。解决用户在 macOS 上快速制作符合社交平台（如微信、飞书）规格 GIF 的痛点，强调“即拖即用”和“体积优化”。

## 2. 核心功能 (Functional Requirements)

### 2.1 格式支持
- **输入**: `.mp4`, `.mov`, `.png`, `.jpg`, `.jpeg`。
- **输出**: 优化的 `.gif`。

### 2.2 转换逻辑
- **视频转 GIF**:
  - 自动调整分辨率（默认宽度 240px，保持纵横比）。
  - 帧率控制（默认 10-12 FPS）。
  - 调色板优化（防止颜色断层）。
- **图片转 GIF**:
  - 静态图单帧转换。
  - 支持将图片序列合成为动态 GIF。

### 2.3 交互体验 (UX)
- **拖拽支持 (Drag & Drop)**: 核心交互方式，支持从 Finder 或桌面直接拖入文件。
- **一键定位**: 转换完成后，自动在 Finder 中高亮显示结果文件。
- **窗口置顶**: 方便在多个文件夹间操作时应用不被遮挡。

## 3. 技术规格 (Technical Specifications)

### 3.1 技术栈建议
- **语言**: Python 3.10+ (配合 `moviepy`, `Pillow`) 或 Swift (配合 `AVFoundation`)。
- **GUI**: Tkinter (配合 `tkinterdnd2-universal`) 或 SwiftUI。
- **关键依赖**:
  - `FFmpeg`: 用于高性能视频解码。
  - `ImageIO` (Swift) / `Pillow` (Python): 用于 GIF 编码。

### 3.2 性能目标
- **转换速度**: 10秒内的短视频转换时间应控制在 3秒内。
- **体积控制**: 默认输出文件大小尽量控制在 5MB 以内。

## 4. 界面设计 (UI Design)
- **主区域**: 虚线边框拖放区。
- **参数区**: 
  - 宽度输入框 (默认 240)。
  - 帧率滑块/输入框 (默认 12)。
- **状态栏**: 显示当前任务进度和成功提示。

## 5. 路线图 (Roadmap)
- **V1.0 (MVP)**: 实现拖拽单视频转换及 Finder 定位。
- **V1.1**: 增加转换进度条及批量处理支持。
- **V1.2**: 集成 `gifsicle` 进行极致压缩，支持自定义剪辑时长。
