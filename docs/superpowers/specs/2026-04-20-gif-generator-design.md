# GIF Generator for macOS - 技术设计规格说明书 (Technical Spec)

## 1. 项目愿景
打造一款 macOS 平台上最轻量、最快速的“即拖即用”表情包制作工具。

## 2. 核心技术栈
- **开发语言**: Swift 5.10+
- **框架**: SwiftUI (macOS 14.0+)
- **核心库**: 
  - `AVFoundation`: 用于视频解析与帧提取。
  - `ImageIO`: 用于 GIF 编码。
  - `UniformTypeIdentifiers`: 处理拖拽文件的类型识别。

## 3. 架构设计 (MVVM)

### 3.1 Model 层
- `GifSettings`: 存储用户配置（宽度、FPS）。
- `ConversionTask`: 描述单个转换任务的状态（输入 URL、输出 URL、进度、错误信息）。

### 3.2 ViewModel 层 (`ConversionViewModel`)
- 负责监听拖拽事件。
- 管理 `AVAssetImageGenerator` 的生命周期。
- 驱动 UI 进度条更新。

### 3.3 View 层
- `ContentView`: 主容器。
- `DropZoneView`: 处理拖放逻辑的核心 UI。
- `SettingsView`: 悬浮参数配置界面。

## 4. 关键算法逻辑

### 4.1 视频转 GIF 流程
1. **加载资产**: 使用 `AVURLAsset` 加载源文件。
2. **提取帧**:
   - 计算提取时间间隔 `1.0 / fps`。
   - 使用 `AVAssetImageGenerator` 的 `generateCGImagesAsynchronously` 批量获取 `CGImage`。
3. **图像处理**:
   - 保持纵横比，将每帧缩放至目标宽度。
4. **GIF 编码**:
   - 使用 `CGImageDestinationCreateWithURL` 创建目标文件。
   - 设置 `kCGImagePropertyGIFDictionary` 控制循环次数（0 为无限循环）。
   - 为每一帧设置 `kCGImagePropertyGIFDelayTime`。
   - 在 `autoreleasepool` 中完成添加帧的操作以节省内存。

### 4.2 文件处理策略
- **输出路径**: 默认与源文件同目录。
- **命名冲突**: 若 `file.gif` 已存在，依次尝试 `file_1.gif`, `file_2.gif` 等。
- **完成后操作**: 调用 `NSWorkspace.shared.activateFileViewerSelecting([outputURL])` 高亮显示结果。

- **窗口特性**: 
  - 支持“始终置顶” (Always on Top)。
  - 窗口边框可增加微弱的绿色呼吸发光效果 (Glow)。
- **视觉反馈**:
  - **初始状态**: 绿色虚线框 + 提示文字 `[ DRAG & DROP FILE HERE ]`。
  - **进行中**: 
    - 矩形方块组成的进度条（如 `[■■■■□□□□] 50%`）。
    - 增加类似终端滚动的日志输出（如 `> EXTRACTING FRAMES...`）。

  - **失败**: 红色等宽文字提示 `[!] ERROR_CODE_01: FAILED_TO_EXTRACT`。
